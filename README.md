# Chat-App-ReactJS
Create a simple dynamic banking website.
Flow: Home > View Customers > Transfer Money > Transaction History
No Login Page. No User Creation. Only transfer of money between multiple users.

### Steps to Run:
1. Clone This App
2. npm install
3. npm run start
